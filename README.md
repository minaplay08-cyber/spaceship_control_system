# Лабораторная работа №7: Система безопасности космического корабля

Разработано в рамках курса по ООП (Python).  
Цель: реализация классов с приватными атрибутами, геттерами/сеттерами и защитой доступа.

## Файлы проекта
- `space_section.py` — класс `SpaceSection` с приватными полями (`__oxygen_level`, `__temperature`, `__access_code`)
- `spaceship.py` — класс `Spaceship`, управляющий 3 отсеками
- `main_menu.py` — интерактивное меню управления
- `Screenshots/` — скриншоты выполнения программы

## Особенности реализации
- Геттеры и сеттеры для безопасного доступа к параметрам отсека
- Метод `emergency_report()` — доступ только при вводе пароля капитана (`admin123`)
- Проверки диапазонов значений (кислород: 0–100%, температура: −50…+50°C, код доступа: 4 цифры)
- Интерактивное меню с выбором действий
